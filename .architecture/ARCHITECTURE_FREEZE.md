# Architecture Freeze

Versão:

4.0

Status:

APPROVED

Architecture:

FROZEN

Após múltiplas auditorias independentes a arquitetura da V4 foi considerada madura.

A implementação deverá seguir exatamente esta arquitetura.

Mudanças estruturais só poderão ocorrer mediante nova RFC.

Alterações locais de implementação não caracterizam mudança arquitetural.

Toda quebra de princípios exige aprovação arquitetural.
